# Compressor-de-Imagens-Python-Pillow-
Script em Python que processa imagens em lote, redimensiona pela largura definida, converte formatos (JPEG, PNG, WEBP) e aplica compressão com qualidade configurável. Usa Pillow para manipulação das imagens e salva os resultados em uma pasta de saída com relatório de redução de tamanho.
